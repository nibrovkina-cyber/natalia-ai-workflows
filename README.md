# natalia-ai-workflows

Free AI agents and skills for small business owners. Built with Claude Code.

**The problem:** You spend hours on tasks that should take minutes. Emails, reports, reviews, scheduling, expenses — all manual, all repetitive, all stealing time from growing your business.

**The solution:** Drop-in AI skills that handle the boring stuff. Clone, configure in 5 minutes, save 1-3 hours daily.

---

## Skills

| Skill | What it solves | Time saved |
|---|---|---|
| [AI Business Assistant](ai-business-assistant/) | Morning routine: emails, calendar, tasks, briefing | 1-2 hours/day |
| [Content Factory](content-factory/) | Content creation: trends, posts, repurpose across platforms | 3-5 hours/week |
| [WB Review Responder](wb-review-responder/) | Marketplace reviews: auto-drafts responses in brand voice | 2-3 hours/day |
| AI CFO *(coming soon)* | Financial reports, expense tracking, cash flow alerts | 4-6 hours/week |
| AI Lead Qualifier *(coming soon)* | Qualify inbound leads, score by ICP fit, draft follow-ups | 1-2 hours/day |

## Quick Start

```bash
# 1. Clone this repo
git clone https://github.com/nataliyabrovk/natalia-ai-workflows.git

# 2. Copy a skill to your Claude Code project
cp -r natalia-ai-workflows/ai-business-assistant ~/.claude/skills/

# 3. Tell Claude about your business
# Open Claude Code and describe your business context

# 4. Use it
# "Morning briefing"
# "Draft replies to my emails"
# "Weekly review"
```

# Contributing

## What's welcome

Bug reports with specifics. "The scraper returns 0 lots when I set region to Ekaterinburg" is useful. "It doesn't work" is not.

Skill requests if you actually tried the agent and something was missing. "I need the review responder to handle Ozon, here's the API difference" is great. "Add support for all marketplaces" is too vague.

Edge case discoveries. If the agent does something unexpected with a real input, open an issue with the input (anonymized) and what you expected.

## What's not welcome

Generic "best practices" PRs that restructure code without solving a problem.

"Add feature X" without explaining what business problem it solves and why the current setup doesn't cover it.

AI-generated issues or PRs that are clearly copy-pasted from ChatGPT without any context.

## Response time

I work on this between feedings and LEGO. Expect 3-5 day turnaround on issues. PRs I look at on weekends.
